### Installing

Clone the project:

```bash
$ git clone https://github.com/md-nasir/monerkotha.git
```

Update composer:

```bash
$ update composer
```

####### Find this values and replace with your puhser account data :

```bash
XXX_PUSHER_APP_ID
XXX_PUSHER_APP_KEY
XXXP_USHER_APP_SECRET
```

Replace .env.monerkotha to .env and configure database and run 

```bash
$ php artisan migrate
```
